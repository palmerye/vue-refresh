# Vue2.0 不断完善版

> 不断完善中的vue2.0个人项目，通过记录一些commit msg，留下一些进步的痕迹. 

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
