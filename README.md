# Vue2.0 小项目系列

> 视频相关的小项目，涉及视频点播，视频上传，用户权限...

> 新环境要操作一段时间dom，有时候还用jQuery，所以还是要经常敲敲MV*框架的代码，技术保鲜...

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
